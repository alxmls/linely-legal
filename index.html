1. Qui sommes-nous ?

LINELY est une application de journal intime numérique développée par Alexandre Xmls.
Contact : support@linely.fr

2. Données collectées

Que tu nous fournis :

  • Compte : e-mail, prénom, nom d’utilisateur
  • Profil : photo de profil, biographie (optionnels)
  • Journal : textes, humeurs, photos, enregistrements audio
  • Contenu social : posts partagés, réactions, encouragements

Automatiquement : streak, nombre d’entrées, version app, type d’appareil, token de notification push.

Ce que nous ne collectons PAS : localisation GPS, contacts téléphoniques, données de navigation externe, données biométriques.

4. Où sont stockées tes données ?

Tes données sont hébergées sur Supabase (serveur EU West — Paris, France), dans l’Union Européenne, conformément au RGPD. Les abonnements sont gérés par RevenueCat (conforme RGPD).

5. Partage des données

Nous ne vendons jamais tes données. Nous les partageons uniquement avec :

  • Supabase — hébergement base de données (EU)
  • RevenueCat — gestion des abonnements
  • Apple — Sign in with Apple et notifications push
  • Autorités compétentes — si requis par la loi

6. Durée de conservation

  • Compte actif : jusqu’à suppression
  • Après suppression : données effacées sous 30 jours
  • Données de facturation : conservées 10 ans (obligation légale)

7. Tes droits (RGPD)

Pour exercer tes droits : support@linely.fr
Réclamation : CNIL — www.cnil.fr • ICO (UK) — www.ico.org.uk

8. Droits CCPA (Californie)

Si tu résides en Californie, tu peux demander quelles données nous détenons et leur suppression. Nous ne vendons pas tes données. Contact : support@linely.fr

9. Sécurité

  • Chiffrement en transit (HTTPS/TLS)
  • Chiffrement au repos (Supabase)
  • Row Level Security — tes entrées ne sont accessibles que par toi
  • Face ID / Touch ID traité localement sur l’appareil uniquement

10. Âge minimum

LINELY est destiné aux personnes âgées de 13 ans et plus.

11. Modifications

Toute modification substantielle sera notifiée via l’application.
